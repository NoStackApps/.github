# NoStackApps

**Build real apps using only free-tier tools. No infrastructure costs. No vendor lock-in.**

We build and share open-source tools and apps powered by Google Apps Script, Google Sheets, Cloudflare Workers, Vue.js, and other free-tier services.

## The Stack

```
Frontend:  Vue.js (free)
Backend:   Google Apps Script (free)
Database:  Google Sheets (free)
CDN/Proxy: Cloudflare Workers (free tier)
Hosting:   GitHub Pages / Cloudflare Pages (free)
```
